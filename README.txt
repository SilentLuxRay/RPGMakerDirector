RPGMakerDirection — Editor di dialoghi per RPG Maker MZ
--------------------------------------------------------

Autore: SideSilent
Versione: 1.0
Data: Novembre 2025

Descrizione:
Questo strumento consente di creare, modificare e salvare battute di dialogo in stile RPG Maker.
Ogni battuta può includere un nome personaggio, un faceset (immagine PNG) e un testo con formattazione HTML.

Funzionalità principali:
- Inserimento di battute con nome, faceset e dialogo
- Modifica e cancellazione tramite menu contestuale
- Salvataggio e caricamento del copione in formato JSON
- Evidenziazione visiva delle battute modificate
- Supporto per immagini PNG e formattazione HTML

Requisiti:
- Sistema operativo Windows (x64)
- Nessuna installazione necessaria (versione standalone)
- Tutti i file necessari sono inclusi nella cartella

File inclusi:
- RPGMakerDirection.exe → programma principale
- System.Text.Json.dll → libreria per salvataggio JSON
- notifica.wav → suono di conferma (opzionale)
- copione.json → file di esempio (opzionale)
- immagini\ → cartella per faceset PNG (opzionale)

Istruzioni:
1. Avvia RPGMakerDirection.exe
2. Inserisci nome personaggio, dialogo e carica un faceset
3. Clicca su "Invia" per aggiungere la battuta
4. Usa il tasto destro del mouse su una battuta per modificare o eliminare
5. Salva il copione con "Salva Copione"
6. Carica un copione esistente con "Carica Copione"

Note:
- Il file JSON viene salvato nella stessa cartella del programma
- Le immagini devono essere in formato PNG
- Il suono di notifica è facoltativo e può essere rimosso

Grazie per aver utilizzato RPGMakerDirection!
